The session

The session will look at:
  1. Building a ML classification model from scratch
  2. Working with the Recipes package to preprocess your data ready for a predictive model
  3. Approaches to data splitting and cross validation with rsample
  4. Building a baseline model with Parsnip
  5. Improving this with hyperparameter tuning with the Tune package
  6. Evaluating your model with Yardstick

The aim is to teach you a basic understanding of how the functionality works for the package and present a couple of real life use cases for how to quickly develop scalable models with this framework. I will use NHS datasets to work on the particular problems in question.
